# Sanutem

## About
Sanutem is a system dedicated to health, both human and animal. It allows professionals to offer their services through an online platform, providing the user with a simple and easy way to set up appointments and observe medical tests.

## LOGIN

Para ingresar al Sistema, seleccionar el botón de‘Login’que se encuentra
en el margen superior derecho.

El Sistema le pedirá ingresar **_Username_** y **_Password_** .
En caso de no contar con uno, puede registrarse con el botón ‘SignUp’

Luego de completar los campos solicitados, seleccionar el botón ‘Sign In’.

## SIGN UP

Para registrarse en el Sistema, seleccionar el botón de ‘Sign Up’ que se
encuentra en el margen superior derecho.


El sistema le pedirá completar ciertos datos:

Dependiendo del perfil que seleccione para crear su cuenta, se le pedirá ingresar información adicional.


#### 1- PERFIL PACIENTE

#### 2- PERFIL PROFESIONAL


### ACTIVACIÓN DE CUENTA

Para concretar la creación de la cuenta, el usuario deberá activarla
ingresando al link que se le enviará por correo electrónico una vez que
finalice el registro.

El sistema informará por pantalla que la activación fue exitosa.


## MENÚ

Todos los usuarios, sin importar el perfil que tengan, tendrán acceso a un
menú principal con 3 opciones:

- Profile
- Settings
- Logout

Para acceder a las mismas, deberá hacer clic sobre el nombre del usuario
que aparecerá en el margen superior derecho.


## PERFIL PACIENTE

Dentro del menú Profile el paciente podrá encontrar varias opciones.

### Search Professionals

Esta opción le permitirá al usuario poder realizar la búsqueda de un
profesional. Para ello se le pedirá que seleccione una especialización, una
provincia y el nombre de la cobertura médica que posee.


Una vez completados los datos, deberá tocar el botón ‘Search’.

Finalmente, se mostrará en pantalla el resultado de la búsqueda.


### Add Pet

El usuario contará con la posibilidad de cargar la información de su
mascota.


Una vez que el formulario se encuentre completo, deberá tocar el botón
‘Register’ para completar la operación.


### Manage Pets

Esta opción nos permitirá ver la información de nuestras mascotas, así
como también editarla o eliminarla.

### Manage Pets - Update

Para editar la información de su mascota deberá tocar el botón ‘Update’.


Se mostrará en pantalla un formulario con la información para ser editada.
Para finalizar la operación seleccionar el botón ‘Update’.

### Manage Pets - Delete

Para eliminar la información sobre su mascota deberá tocar el

botón ‘Delete’.


### Attach Medical Tests

El usuario podrá cargar en el sistema todos sus estudios médicos en
formato .pdf. Para poder realizarlo deberá seleccionar el botón
‘Seleccionar Archivo’.

Automáticamente se abrirá una ventana que le permitirá seleccionar un
archivo desde su pc.


Una vez seleccionado el archivo, deberá tocar el botón ‘Upload’.

Al finalizar la operación, el sistema mostrará por pantalla que el archivo se
subió de manera exitosa.


## PERFIL PROFESIONAL

Dentro del menú Profile el profesional podrá encontrar varias opciones.


### Link Receptionist

El profesional podrá vincular su cuenta con el ID de un recepcionista. Esto
le dará permisos al recepcionista para administrar el calendario y los
turnos del profesional.

Para completar la acción deberá completar el campo ‘ID Receptionist’ con
el número identificador del recepcionista. La operación finalizará al tocar el
botón ‘Link’.


### View Patient’s History - Search

El profesional tendrá la oportunidad de visualizar el historial médico de sus
pacientes.

Primero deberá seleccionar el usuario del paciente cuya información desea
ver. Para finalizar la operación deberá tocar el botón ‘Search’.


El sistema mostrará por pantalla una tabla con el historial médico del
paciente.

### View Patient’s History – Add

El profesional podrá agregar información al historial clínico del paciente,
para ello deberá seleccionar el usuario del paciente y luego tocar el botón
‘Add’.


Inmediatamente se desplegará una pantalla que le permitirá cargar la
fecha y los detalles de la consulta.

Para finalizar la operación deberá tocar el botón ‘Add’.


Finalmente se podrá visualizar que la información se ha guardado de
manera correcta.

El profesional sólo tendrá acceso al historial clínico de los pacientes que
hayan reservado un turno con él.


### View Calendar

El profesional tendrá acceso a la información de los turnos que ya han sido
reservados por los pacientes.


## PERFIL RECEPCIONISTA

Dentro del menú Profile el recepcionista podrá encontrar varias opciones.

En esta pantalla se podrá visualizar el ID del recepcionista. Este ID es el
que luego utilizará el profesional para vincular su cuenta con la del
recepcionista.

Para poder visualizar con información a las pantallasAdd Availabilityy
View Appointmentses necesario que, con anterioridad, algún profesional
realice una vinculación con la cuenta del recepcionista.


### Add Availability

En esta pantalla el recepcionista podrá agregar la disponibilidad horaria de
atención del profesional por mes.


Una vez seleccionada la información, se podrá guardar tocando el botón
‘Submit’.


### View Appointments

El recepcionista tendrá acceso a la información de los turnos que ya han
sido reservados por los pacientes.

El mismo contará con la posibilidad de cancelar los turnos. Este proceso
se realiza tocando el botón ‘Cancel’.


## Modify Profile – (para todos los perfiles)

Todos los usuarios (pacientes, profesionales y recepcionistas)

podrán modificar cierta información de su perfil.

El usuario, contraseña y número de documento no podrán ser

modificados.


## DELETE ACCOUNT

Para eliminar su cuenta deberá seguir los siguientes pasos:

```
1- Ingresar a ‘Settings’.
```
```
2- Hacer clic sobre el botón ‘Delete Account’.
```

## LOGOUT

Para cerrar sesión deberá tocar el botón‘Logout’que se encuentra en el
menú desplegable en el margen superior derecho.
